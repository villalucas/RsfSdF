Initialise la configuration de l\textquotesingle{}interface SPI1. 

Cette fonction configure l\textquotesingle{}interface SPI1 ainsi que les broches associées pour l\textquotesingle{}utilisation du module SPI. Elle initialise les registres de configuration pour établir les paramètres suivants \+:
\begin{DoxyItemize}
\item Utilisation de PB6 comme broche de sélection du périphérique (CS).
\item Utilisation de PA5, PA6 et PA7 comme broches d\textquotesingle{}interface SPI1.
\item Configuration des broches en mode alternatif (AF) pour l\textquotesingle{}utilisation de l\textquotesingle{}interface SPI.
\item Configuration du SPI1 en mode maître avec une vitesse de transmission de 6 MHz.
\end{DoxyItemize}

\begin{DoxyNote}{Note}
Cette fonction suppose que les registres RCC, GPIOB, et GPIOA ont déjà été initialisés correctement. 

Les détails spécifiques concernant les registres sont documentés dans le code source. 
\end{DoxyNote}
